# 프로비저닝	
## 프로비저닝 목록 조회	
- 프로비저닝된 충전기 목록을 조회할 수 있다
- 프로비저닝 충전기란, 설치기사가 충전소에 설치하러 간 인증된 충전기를 말한다. 
- 프로비저닝 충전기로 등록되어야만 CSMS는 정상등록된 충전기로 인식한다

1) 조회정보
   - 충전기id
   - key
   - value
   - 등록일
   - 등록상태(정상/에러)
   - 에러설명"
## 프로비저닝 정보 검색	
- 프로비저닝 정보중 원하는 충전기 정보가 등록되어있는지 검색할 수 있다
- 사용자가 입력한 검색어는 아래 검색항목중 일부가 매칭되어도 검색결과에 포함시켜 준다

1) 검색되는 항목
   -  충전기 id
   - key
   - value"
## 프로비저닝 등록	
- 하나의 충전기를 등록할 수 있다

1) 등록항목
   - 충전기id
   - key
   - value"
## 프로비저닝 일괄등록	
- 여러 대의 충전기를 csv파일을 이용하여 일괄등록할 수 있다
## 프로비저닝 일괄등록 예제파일 다운로드	
- 프로비저닝 일괄등록시 사용할 수 있는 csv파일의 예제파일을 다운로드할 수 있다
## 프로비저닝 변경	
- 프로비저닝된 충전기의 한 항목을 지정하여 정보를 변경할 수 있다

1) 변경가능한 항목
   - 충전기 id
   - key
   - value"
	프로비저닝 삭제	등록된 프로비저닝 정보중 선택하여 삭제할 수 있다

# 장애 로그	
## 장애 로그 조회	
- 장애발생 로그를 아래의 기준으로 검색할 수 있다

1) 검색조건
   - 조회 시작일/종료일 : 필수입력정보
   - 파트너 id
   - 충전소명
   - 충전기 
   - 상태 : 전체 | 접수 | 조치중 | 완료
   - 키워드 : 로그중에 포함된 단어(부분 검색 적용)
2) 검색결과 정보
   - 장애로그 생성일
   - 파트너
   - 충전소
   - 충전기id
   - 장애유형 : 어떤 것이 있나?
   - 설명
   - 보고자
   - 상태
   - 상태변경"
## 장애등록	
- 장애를 수동으로 등록하는 메뉴

1) 등록정보
   - 충전기
   - 장애유형
   - 설명"
